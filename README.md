# 3DPlatformer

Download for PC - https://tinyurl.com/57df9sjc

Controls:

  Keyboard:
  
  WASD - Move  
  Space - Jump   
  Q - Timelapse  
  LeftShift - Shot jump  
  Ctrl+F4 - Quit (Sorry)   
  
  Mouse:
  LMB - Shot;
  RMB - Hook
